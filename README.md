Весь код написан на языке C#.

---

## 🛠 Технологический стек

- **Язык программирования:** C#  
- **Платформа:** .NET (Core / Framework)  
- **IDE:** Visual Studio 2022  
- **Инструменты:** NuGet, MySQL, MSBuild  

---

## 📂 Структура репозитория

Каждая папка — отдельный проект.

| Папка       | Тема работы                  | Ключевые технологии                    |
|-------------|------------------------------|----------------------------------------|
| `PR-1`      | Линейные алгоритмы           | Примитивные типы, ввод/вывод в консоль |
| `PR-2`      | Условные операторы и циклы   | Логика ветвления, `switch`, `for`/`foreach` |
| `PR-3`      | Работа с массивами и коллекциями | `List<T>`, `Array`, работа со списками |
| `PR-4`      | Объектно-ориентированное программирование | Классы, интерфейсы, наследование |
| `PR-5`      | Обработка исключений и файлы | `try-catch`, `System.IO`               |

---

## 🚀 Запуск проектов

Чтобы запустить любую из работ локально, убедитесь, что у вас установлен **.NET SDK**.

1. Клонируйте репозиторий:
```bash
  git clone https://github.com/kenn1d/MDK.05.02_master-repo.git
   ```
2. Перейдите в папку нужного проекта:
```bash
  cd praktika(№работы)
  ```
3. Запустите проект через терминал:
```bash
  dotnet run
  #Или просто откройте файл .sln в Visual Studio и нажмите F5.
```
